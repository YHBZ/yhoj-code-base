# yhoj-code-base
YHOJ致力于为OIer们提供更好学习资源！


这是YHOJ的代码库。

它只为开发与发布而提供。

每个人都能查看并且复制它，前提是遵循GPLv3.0中的规定。

每个PUSH都在DEVELOP上，而不是RELEASE。RELEASE只能接收DEVELOP的拉取请求以更新自己（fork者们的拉取请求在DEVELOP上）。 

YHOJ一定会走向胜利！
